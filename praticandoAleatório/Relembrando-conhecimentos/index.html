Método Map():

- O método map é um método de array, onde ele cria um novo array com a mesma quantidade de elementos que o array original porem com os novos elementos tranformados. veja no exemplo abaixo:

const numbers = [2, 3, 5, 12]

const resultNumbers = numbers.map(item => item * 2) // output = [10, 15, 25, 60]
